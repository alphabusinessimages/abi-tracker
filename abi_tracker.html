⚠ Firebase setup required for live sync

To enable live updates across your team, follow these steps (takes about 5 minutes, completely free):

  1. Go to console.firebase.google.com and sign in with a Google account
  2. Click "Add project" → name it "ABI Tracker" → Create project
  3. In the left sidebar click "Realtime Database" → "Create database" → choose "Start in test mode" → Enable
  4. In the left sidebar click the gear icon ⚙ → "Project settings" → scroll to "Your apps" → click </> (Web icon)
  5. Register your app (any name) → copy the firebaseConfig object values
  6. Open this HTML file in a text editor → find the FIREBASE CONFIG section → paste your values
  7. Save and re-open. Share the updated file with your team. Everyone will see live updates!

In the meantime, the tracker works normally and saves locally in your browser.
